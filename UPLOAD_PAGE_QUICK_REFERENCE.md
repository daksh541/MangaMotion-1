# Upload Page - Quick Reference

## Features at a Glance

### Main Upload Zone
- **Drag & Drop**: Drag manga pages directly onto the zone
- **Click to Browse**: Click "Select Files" button to open file picker
- **Supported Formats**: JPG, PNG, WEBP
- **Max File Size**: 50MB per file

### File Management
- **View Files**: Selected files appear in a list below
- **File Info**: Name and size shown for each file
- **Remove File**: Click the X button to remove individual files
- **Clear All**: Use "Clear All Files" button to reset selection

### Upload Process
1. Select files (drag-drop or click button)
2. Review files in the preview area
3. Click "Continue to Review" button
4. Watch progress bar as files upload
5. Automatically navigate to next step

### Demo Mode
- Click "Try a Demo Project" to test without uploading
- Demo files are pre-loaded and ready
- Full workflow available with demo files
- Great for testing the entire process

### Pro Tips
- Click "Pro Tips for Best Results" to expand
- Learn best practices for uploads
- Quality recommendations
- File naming conventions
- Reading order guidelines

## Keyboard Shortcuts
- **Enter**: Submit upload (when files selected)
- **Tab**: Navigate between elements
- **Space**: Activate buttons

## File Size Display
- Sizes shown in MB
- Total size calculated automatically
- Updated in real-time as files added/removed

## Status Messages
- "Preparing files..." - Initial setup
- "Uploading file X of Y" - Current upload
- "Processing files..." - Final processing
- "100%" - Upload complete

## What Happens Next
1. Files are uploaded to server
2. Project data saved
3. Redirected to detection.html
4. Can review and edit files
5. Start animation process

## Tips for Best Results
- Use high-quality images (1000px+ minimum)
- Name files in order (page01.jpg, page02.jpg)
- Upload in reading order (left to right, top to bottom)
- Keep file sizes reasonable
- Use consistent image dimensions

## Troubleshooting

### Files Not Showing?
- Check file format (JPG, PNG, WEBP only)
- Verify file size (max 50MB)
- Try refreshing the page

### Upload Stuck?
- Check internet connection
- Try with fewer files
- Clear browser cache
- Try different browser

### Demo Not Loading?
- Click "Try a Demo Project" button
- Wait for loading animation
- Demo files will appear in list
