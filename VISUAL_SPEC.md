## Animation Specifications

### Neon Glow (Social Icons)
```
Duration: 200ms
Easing: ease-out
Keyframes:
  0%:   box-shadow: 0 0 8px rgba(168, 85, 247, 0.4)
  100%: box-shadow: 0 0 12px rgba(168, 85, 247, 0.6)
```

### Pulse Soft (Input Focus)
```
Duration: 400ms
Easing: ease-in-out
Keyframes:
  0%:   transform: scale(1)
  50%:  transform: scale(1.01)
  100%: transform: scale(1)
```

### Underline Fade (Link Hover)
```
Duration: 300ms
Easing: ease-out
Direction: Left to Right
Keyframes:
  0%:   width: 0, opacity: 0
  100%: width: 100%, opacity: 1
```

### Chevron Rotate (Accordion)
```
Duration: 300ms
Easing: ease-out
Keyframes:
  Closed:  rotate(0deg)
  Opened:  rotate(180deg)
```

## Responsive Behavior

### Desktop (1024px+)
- 4-column grid layout
- Horizontal social icons
- Side-by-side form (email + button)

### Tablet (768px - 1023px)
- 2-column grid layout
- Horizontal social icons
- Stacked form (email full-width, button below)

### Mobile (< 768px)
- Single column (accordion)
- Vertical social icons (2x2 grid)
- Stacked form
- Larger touch targets (48px)
- Chevron indicators for expandable sections

## Micro-Interactions Timeline

### Social Icon Hover
```
0ms:    User hovers
0-200ms: Background color shift, scale increase, glow appear
200ms:  Animation complete, element stays in hover state
```

### Link Underline
```
0ms:    User hovers
0-300ms: Underline grows from left to right with gradient
300ms:  Animation complete, underline stays visible
```

### Input Focus
```
0ms:    User focuses
0-400ms: Border color change, glow appear, pulse animation
400ms:  Animation complete, element ready for input
```

### Form Submission
```
0ms:    User clicks submit
0-100ms: Button scale down (0.98)
100ms:  Button returns to normal scale
200ms:  Success message appears (if applicable)
3000ms: Success message fades
```
